## Run Locally

Clone the project

```bash
  git clone https://github.com/gabbar-singhh/willyoubemyvalentinee.git
```

Go to the project directory

```bash
  cd willyoubemyvalentinee
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
